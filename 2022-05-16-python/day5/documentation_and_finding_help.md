# getting help / documentation

## inside Python

- `help()`
- `dir()`

e.g.:

```py
# show complete documentation
help(str)

# show all method names
dir(str)
# [... encode, endswith, expandtabs, find, format, ...]

dir(str.find)
# ...
```
